REST API

SCOP REST API Endpoints

Resource Description
term/:scop_id retrieve all information about the term
children/:scop_id list all direct descendants of the term
parents/:scop_id list all direct ancestors of the term
domains/:scop_id list all domains classified at this node
domains_by_uniprot/:uniprot_id list all domains associated with a given UniProt ID
ancestry/:scop_id retrieve the ancestry graph of the term
represented_structures/:scop_domain_id list the structures represented by the domain
